**🏏 IPL Player Statistics Analyzer**

The IPL Player Statistics Analyzer is a desktop-based data visualization project developed using Python. It provides an interactive graphical interface to analyze cricket player performance based on key metrics like runs, wickets, and strike rate.

The application calculates a performance score for each player and presents insights using multiple visualization techniques such as bar charts, radar charts, and comparison graphs.

_🎯 Key Features_

📊 Performance score calculation
🏆 Top player identification
📋 Player rankings
🔍 Search and filter functionality
👥 Player comparison
📈 Multiple chart visualizations
💾 CSV data import/export
🧠 Performance Formula
Performance Score = Runs + (Wickets × 20) + Strike Rate

🛠️ Tech Stack
Python
Tkinter (GUI)
Matplotlib (Visualization)
NumPy
CSV Module
Pillow (PIL)

📊 Visualizations Included
Runs Bar Chart
Performance Score Chart
Radar Chart (Top Players)
Player Comparison Chart
Custom Field Visualization

This project demonstrates how data analytics and visualization techniques can be applied to sports data to generate meaningful insights and 
support decision-making.
