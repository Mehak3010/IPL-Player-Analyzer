# 🏏 IPL Player Statistics Analyzer

![Python](https://img.shields.io/badge/Python-3.x-blue)

![Matplotlib](https://img.shields.io/badge/Visualization-Matplotlib-orange)

A desktop-based data visualization application built using Python to analyze IPL player performance through interactive charts and rankings.

---

## 📌 Overview

The **IPL Player Statistics Analyzer** is designed to help users analyze cricket player performance using key metrics like runs, wickets, and strike rate.

It provides a user-friendly interface where users can:
- Add and manage player data  
- Calculate performance scores  
- Visualize data using charts  
- Compare players  

---

## 🎯 Features

- 📊 Performance Score Calculation  
- 🏆 Top Performer Identification  
- 📋 Player Rankings  
- 🔍 Search & Filter Players  
- 👥 Player Comparison  
- 📈 Multiple Data Visualizations  
- 💾 CSV Import/Export  

---

## 🧠 Performance Formula
- Performance Score = Runs + (Wickets × 20) + Strike Rate
